前言

======================================
==========修改之前需要注意以下几点==========

  1. 反编译的部分需要合并odex
  2. 需要安装 java 并添加变量
  3. 备份好您的系统或者备份好修改部分
  4. 编译系统自带用原签名,其它 apk 用重新签名！
  5. 请用目录方法查看您需要修改的部分！
  6. 部分机型 SystemUI.apk 名字不相同！教程通用
    • LG———–LGSystemUI.apk
    • xiaomi—–MiSystemUI.apk
  7. 下面代码可能部分机型不相同但是差不多！
  8. 工具我使用的是 - APKDB-官网地址

第三方状态栏全透明

  1. 反编译SystemUI.apk
    • 打开\SystemUI\res\values\colors.xml
  2. 搜索：<color name="system_bar_background_semi_transparent">
  3. 得到：<color name="system_bar_background_semi_transparent">#66000000</color>
    这部分是修改状态栏颜色透明度,默认QQ状态栏是有些暗的，照下面修改你喜欢的透明度。
    修改后系统、第三方软件全部根据这个值来调整颜色深度！
    • #66000000 这是系统默认的值,代表不透明占66% 默认值根据机型不同而变化
    • #00000000 这是修改后的值,代表全透明

下拉栏透明

  1. 打开\SystemUI\res\values\colors.xml
  2. 搜索：system_bar_background_transparent

    这部分是修改下拉栏颜色透明度,照下面修改你喜欢的透明度。

    • #66000000 这是系统默认的值,代表不透明占66% 默认值根据机型不同而变化
    • #00000000 这是修改后的值,代表全透明

其余代码根据需要修改

  • <color name="system_primary_color"> //下拉栏快捷功能的下部分背景颜色
  • <color name="system_secondary_color"> //下拉栏快捷功能的上部分背景颜色
  • <color name="notification_material_background_color"> //通知框的背景颜色。

修改字体颜色

======================================
==========修改之前需要注意以下几点==========

  1. 此方法经过我的修改好像并没有什么用,找不到以下代码
  2. 本人使用 LG G3 F460 进行测试没法修改,原生系统不知！
  3. 备份好您的系统或者备份好修改部分
  4. 编译系统自带用原签名,其它 apk 用重新签名！
  5. 把以下颜色代码 修改为你喜欢的颜色 然后保存并退出
  6. 颜色代码查询： 16进制颜色代码对照查询搜索表
  1. 首先反编译framework-res.apk 得到framework-res文件夹！
  2. 修改里面的文件， 找到framework-res.apk\res\values\sXXX.xml S开头的文件
    • < name="bright_foreground_dark">#ffc8c8c8 设置界面 以及对话框顶栏字体颜色
    • < name="bright_foreground_dark_disabled">#80ffffff未启用功能字体颜色（暗色）
    • < name="dim_foreground_dark">#ffbebebe 关于手机里的 型号 版本号字体颜色
    • < name="dim_foreground_dark_disabled">#80bebebe关于手机里的 型号 版本号以及二级目录字体暗颜色
    • < name="hint_foreground_dark">#ff808080 输入框内提示文本颜色
    • < name="dim_foreground_light">#ff323232关机界面 飞行模式等下的二级文本颜色
    • < name="bright_foreground_light">#ff000000 选择列表字体颜色
    • < name="status_bar_text">#ffffffff状态栏字体颜色
    • < name="status_bar_clock_text">#ffbebebe状态栏时钟字体颜色
    • < name="status_bar_event_primary_text">#ffffffff下拉提示栏字体主颜色
    • < name="status_bar_event_secondary_text">#ffb6cad6下拉提示栏二级字体颜色
    • < name="sub_text_">#ff6e9cba全局二级字体颜色
