打开之后输入以下命令安装必要环境。
apt update
apt install python2 //谨记，上面一条是python2 而不是python
apt install git

git clone http://git.clonehttps://github.com/sqlmapproject/sqlmap.git sqlmap //克隆sqlmap到当前目录下的sqlmap目录
clone 完成后输入 ls 可以看到当前目录下多了一个sqlmap目录
cd sqlmap //进入sqlmap目录
python2 sqlmap.py //运行sqlmap~

但是每次都输入cd sqlmap python2 sqlmap.py 非常不方便~

恰巧termux也支持zsh 的包~ zsh配置文件可以帮忙我们把命令缩短，比如我们直接输入sqlmap 就等同于输入 python2 /路径/sqlmap.py

安装zsh

apt install zsh
安装oh-my-zsh

wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh .oh-my-zsh
cp .oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc //然后输入
cd /data/data/com.termux/files/usr/etc/bin/
chsh -s /bin/zsh
termux开一个新窗口就可以看到效果～变成绿色箭头 如图
安卓termux运行sqlmap和termux使用技巧
说明chsh -s /bin/zsh已经生效了~~

现在 我们可以编辑.zshrc 配置文件来设置别名了（把命令缩短）
apt install vim //安装vim编辑器
vim ~/.zshrc
把屏幕划到最底部，添加一行：
alias cls=’clear’
安卓termux运行sqlmap和termux使用技巧
保存退出
提示： termux几个常用的快捷键
音量键上 加e == esc
音量键上 加t == table
音量键下 加c == ctrl c
音量键下 == Ctrl键
知道这几个快捷键，操作vim就不是难事了~
然后termux 新开一个窗口，输入cls 命令 如果清除了屏幕内容，说明我们刚才配置的
alias cls=’clear’ 生效了～意思就是为clear（清除屏幕命令）命令设置一个别名，输入cls等同于输入clear。

好了，那么我能能不能把刚才运行sqlmap的命令缩短呢（python2 /路径/sqlmap.py）？
当然是可以的！
回到编辑 .zshrc文件
vim ~/.zshrc再在后面添加一行
alias sqlmap=’python2 /data/data/com.termux/files/home/sqlma/sqlmap.py’其中，/data/data/com.termux/files/home/sqlma/sqlmap.py 为你的sqlmap.py 所在的路径。（pwd 命令可以查看当前路径）
完成后 直接输入sqlmap 等同于python2 /data/data/com.termux/files/home/sqlma/sqlmap.py，手机上操作也不再有难度。

关于zsh

关于zhs具体可以百度了解，现在说几个小技巧

按照教程操作完成后，你可不需要再用cd 命令进目录 ，可以直接输入目录名称 如当前目录下有sqlmap目录 直接输入sqlmap 即可进入sqlmap 目录（等同于cd sqlmap）。输入.. 直接返回上级目录，输入…直接放回上上级目录….
